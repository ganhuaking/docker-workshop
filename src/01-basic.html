# 基本指令

指令 功能
docker pull 下載映像檔
docker images 看目前有哪些映像檔
docker rmi 刪除映像檔
docker run 建立容器並執行指令
docker start/stop/restart 操作容器
docker ps 看目前有啟動哪些容器
docker rm 刪除容器

# Hello world

$ docker run -d nginx
$ docker images
$ docker ps
$ docker stop
$ docker rm
